U2PDF is a comprehensive suite of PDF tools that run entirely in your browser. Your files are never uploaded to a server, ensuring 100% privacy and security.

## Tools Included
- **Word to PDF / PDF to Word**
- **Text to PDF** (with Rich Text Editor)
- **PDF to Picture** (PNG/JPG/ZIP)
- **Compress PDF, Image, and Video**
- **Merge & Split PDF**
- **Protect & Watermark PDF**
- **Rotate & Page Numbering**

## Development
```bash
npm install
npm run dev
```

## Deployment to GitHub Pages
1. Ensure `base` in `vite.config.ts` matches your repository name.
2. Run `npm run build`.
3. Deploy the `dist` folder to the `gh-pages` branch.
